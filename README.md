# Latex_Templates
一份逐渐改进的Latex模板。

这份模板是从我的集合论报告中独立出来的项目，看上去也算是简洁美观大方，许多同学觉得不错，特此开源。

# Windows

## 工具
* Convert.bat   将目录下png文件转换为eps文件
* Created.bat   实现tex文件到pdf的自动转换
* Clear.bat     清理临时文件

# Linux
* Created.sh    实现tex文件到pdf的自动转换
* Clear.sh      清理临时文件
